## 🚀 كيفية الاستخدام

1. **تشغيل المشروع**
```bash
npm run dev
```

2. **الوصول للوحة التحكم**
```
http://localhost:3000/doctor-dashboard
```

3. **تسجيل الدخول كطبيب**
- استخدم حساب طبيب موجود
- أو أنشئ حساب جديد بدور "doctor"

## 🔧 التخصيص

يمكن تخصيص الألوان والتصميم من خلال:

1. **ملف المتغيرات**
```css
/* في dashboard.css */
:root {
  --primary-color: #D9A741;
  --secondary-color: #E6B84A;
}
```

2. **تخصيص المكونات**
```jsx
// في أي مكون
style={{ 
  backgroundColor: 'var(--primary-color)',
  color: 'white' 
}}
```
