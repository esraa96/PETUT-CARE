# نظام الدعم الفني المحدث - Petut App

## التحديثات المنجزة ✅

### 1. إزالة Bootstrap
- تم إزالة جميع استخدامات Bootstrap من ملفات الدعم الفني
- تم استبدالها بـ Tailwind CSS مع ألوان petut-brown المتسقة

### 2. تحسين التصميم
- **ألوان متسقة**: استخدام ألوان petut-brown في جميع المكونات
- **تصميم عربي**: ترجمة جميع النصوص للعربية
- **تحسين UX**: إضافة مؤشرات للرسائل غير المقروءة
- **تحسين الإحصائيات**: إضافة وصف لكل إحصائية

### 3. مكان الرد على الرسائل
- **للإدارة**: يمكن للإدارة الرد على رسائل العملاء من خلال:
  - صفحة الدعم الفني: `/admin-dashboard/support`
  - الضغط على زر "فتح المحادثة" أو "رد جديد"
  - كتابة الرد في مربع النص والضغط على "إرسال"

- **للعملاء**: يمكن للعملاء إرسال رسائل والرد على الإدارة من خلال:
  - نافذة الدعم الفني في التطبيق
  - كتابة الرسالة والضغط على "إرسال"

## Flow النظام 🔄

### 1. إنشاء تذكرة دعم
```
العميل → إنشاء تذكرة → حفظ في Firebase → إشعار للإدارة
```

### 2. الرد على التذكرة
```
الإدارة → فتح التذكرة → كتابة الرد → إرسال → إشعار للعميل
```

### 3. متابعة المحادثة
```
العميل/الإدارة → إرسال رسالة → تحديث الحالة → إشعار للطرف الآخر
```

## كيفية الاستخدام 📖

### للإدارة
1. اذهب إلى `/admin-dashboard/support`
2. شاهد الإحصائيات في الأعلى
3. اختر التذكرة المطلوبة من الجدول
4. اضغط على "فتح المحادثة" أو "رد جديد"
5. اكتب ردك واضغط "إرسال"
6. يمكن تغيير حالة التذكرة من القائمة المنسدلة

### للعملاء
1. افتح نافذة الدعم الفني
2. اكتب رسالتك
3. اضغط "إرسال"
4. انتظر رد فريق الدعم
5. تابع المحادثة حسب الحاجة

```javascript
{
  id: "ticket_id",

  userName: "اسم المستخدم",
  userEmail: "email@example.com",
  userImage: "profile_image_url",
  subject: "موضوع التذكرة",
  status: "open|in_progress|resolved|closed",
  priority: "low|medium|high|urgent",
  messages: [
    {
      senderId: "sender_id",
      senderName: "اسم المرسل",
      senderRole: "user|admin",
      message: "نص الرسالة",
      timestamp: Date,
      isAdmin: boolean
    }
  ],
  hasUnreadMessages: boolean,
  createdAt: Date,
  updatedAt: Date
}
```
